AEGIS v4.6.2 - OFFLINE INSTALLATION PACKAGE
================================================================================

This directory now contains everything needed to install AEGIS on any
computer without internet access.

QUICK START:

  Windows Users:
    → Double-click: install_offline.bat

  macOS/Linux Users:
    → Run: bash install_offline.sh

WHAT YOU NEED TO KNOW:

1. Python 3.10+ must be installed on the target system
2. 500 MB free disk space required
3. No internet needed for installation
4. Takes 5-10 minutes to complete

FILES IN THIS PACKAGE:

  wheels/                              - All 122 Python packages (398 MB)
  install_offline.bat                  - Windows installation (double-click)
  install_offline.sh                   - macOS/Linux installation (bash)
  DOWNLOAD_COMPLETION_SUMMARY.txt      - This summary
  WHEELS_INSTALLATION_REPORT.md        - Detailed documentation
  OFFLINE_SETUP_QUICK_START.txt        - Quick start guide
  WHEELS_MANIFEST.txt                  - Package list
  requirements.txt                     - Dependencies list

AFTER INSTALLATION:

  Run AEGIS:
    python app.py --debug

  Open in browser:
    http://localhost:5050

NEED HELP?

  See: OFFLINE_SETUP_QUICK_START.txt (easy start guide)
  Or:  WHEELS_INSTALLATION_REPORT.md (detailed documentation)
