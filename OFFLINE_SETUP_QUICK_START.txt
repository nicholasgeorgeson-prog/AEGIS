================================================================================
MANUAL INSTALLATION (ALL PLATFORMS)
================================================================================

For advanced users or if scripts don't work:

1. Open Command Prompt (Windows) or Terminal (Mac/Linux)

2. Navigate to project folder:
   cd C:\path\to\TechWriterReview  (Windows)
   cd /path/to/TechWriterReview    (Mac/Linux)

3. Install packages:
   python -m pip install --no-index --find-links=wheels -r requirements.txt

4. Verify installation:
   python -c "import flask; print('Flask OK')"
   python -c "import pandas; print('Pandas OK')"

5. Start application:
   python app.py

6. Open: http://localhost:5050

COMMON QUESTIONS
================================================================================

Q: Do I need internet to run the installation?
A: No! All packages are pre-downloaded. Install without internet.

Q: Where should I put the wheels folder?
A: Keep it in the same directory as install_offline.bat/install_offline.sh

Q: Can I delete the wheels folder after installation?
A: Yes, after successful installation you can delete it (saves 398 MB)

Q: What if a package fails to install?
A: Note the package name and try installing it manually:
   pip install --no-index --find-links=wheels PACKAGE_NAME

Q: How do I create a virtual environment?
A: macOS/Linux: bash install_offline.sh --venv
   Windows: python -m venv venv

Q: Can I use this on a different computer?
A: Yes! Copy the wheels folder to the new computer and run install script.

Q: How long does installation take?
A: Typically 5-10 minutes depending on disk speed.

Q: Do I need administrator rights?
A: Usually no, unless installing system-level dependencies like Tesseract.

Q: Is the installation reversible?
A: Yes! Uninstall with: pip uninstall -r requirements.txt -y

GETTING HELP
================================================================================

Installation Failed?
1. Check that wheels/ folder exists (should have 122 files)
2. Check Python version: python --version (must be 3.10+)
3. Check disk space: Need 500+ MB available
4. Try manual installation (see MANUAL INSTALLATION section above)

Application Won't Start?
1. Server not running? Run: python app.py --debug
2. Can't access http://localhost:5050? Wait 10 seconds, try again
3. Port 5050 already in use? Kill other processes or change port

Specific Package Missing?
1. Run: pip list | grep PACKAGE_NAME
2. If not listed, reinstall: pip install --no-index --find-links=wheels PACKAGE_NAME
3. Check error messages for dependency issues
